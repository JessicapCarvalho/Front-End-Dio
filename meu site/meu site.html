Intrudução ao HTML

(voltar)

A Linguagem de Marcação de Hipertexto (HTML) é a linguagem principal da web. Entretando não é uma linguagem de programação. É um protocolo usado para desenvolver paginas na internet.

Conhecida em inglês como Hypertext Markup Language, ela permite que os desenvolvedores construam a estrutura e desenhem como os elementos são exibidos na pagina, como texto, hiperlinks, listas e arquivos.

O html foi criado com o objetivo de facilitar o desenvolvimento de sites e blogs.

Estrutura basica do HTML

(voltar)

Para poder criar ou editar um documento HTML, o único recurso que você precisa é de um editor de texto, como o próprio bloco de notas. Porém para desenvolver de forma mais profissional recomenda-se que você utilize programas como VS code, Visual Studio, Intellij, entre outros.

A estrutura de um HTML é composta por: aberturas e fechamentos de chaves >, contendo dentro delasa os elementos como as tags. o Corpo do HTML é estruturado dentro da chave body , todo o conteudo descrito dentro dessa chave é o que aparecera para o usuario final.

Tags

(voltar)

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros. Além de tags de estruturas podemos encontrar tags de personificação da pagina como os elementos utilizados para editar textos dentro do html.

Textos no HTML

(voltar)

Os textos em HTML podem ser descritos como tudo o que você consegue ler na pagina, para criar textos são utilizadas as Tags como: p para paragrafos, as tags de tamanho que variam de h1 até h6, alem disso pode-se realizar outras edições como colocar o texto em negrito utilizando a tag strong ou em italico com a tag i

Listas no Html.

(voltar)

Muito do conteúdo da web é de listas e o HTML tem elementos especiais para elas. Listas de marcação sempre consistem em pelo menos 2 elementos.

Os tipos mais comuns de lista são ordenadas e não ordenadas: Listas não ordenadas são para listas onde a ordem dos itens não importa, como uma lista de compras, por exemplo. Essas são envolvidas em um elemento ul. Listas Ordenadas são para listas onde a ordem dos itens importa, como uma receita. Essas são envolvidas em um elemento ol.
